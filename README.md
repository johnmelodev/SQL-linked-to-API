Verbs and HTTPS Structure:

GET - Retrieve existing data
POST - Send data
PUT - Update existing data
DELETE - Delete data

Status codes:

1xx: Information
2xx: Success
3xx: Redirection
4xx: Client Error
5xx: Server Error

Need to install:
pip3 install requests
